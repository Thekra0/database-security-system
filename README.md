# Secure SkinCare Database

A database administration and security project that demonstrates how to design and secure a MySQL database for a skincare company by protecting sensitive employee information.

## Features

- Employee and salary database design
- Role-Based Access Control (RBAC)
- User privilege management
- Password hashing using SHA-256
- AES encryption for sensitive salary data
- Secure data retrieval using decryption
- Database security testing

## Technologies

- MySQL
- SQL
- AES Encryption
- SHA-256 Hashing
- Database Administration
- Role-Based Access Control (RBAC)

## Security Features

- Encrypted salary records using AES
- Hashed user passwords with SHA-256
- User roles and privilege management
- Foreign key constraints
- Secure database structure

## Learning Outcomes

- Database design and normalization
- Database security best practices
- Access control implementation
- Data encryption and hashing
- MySQL administration
